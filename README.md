# Portfolio Website Template

A modern, clean portfolio website template built with Next.js and shadcn. Perfect for developers, designers, and professionals who want to showcase their work, skills, and experience.

## Features

- **Personal Introduction** - A dedicated hero section to introduce yourself and your professional background
- **About Section** - Share your story, background, and what makes you unique
- **Resume/CV** - Display your work experience, education, and skills in a clean, organized layout
- **Projects Portfolio** - Showcase your previous work and case studies with detailed project pages
- **Contact** - Easy ways for visitors to get in touch with you

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser and go to `http://localhost:3000`

## Customization

Edit the content in the relevant component files to add your personal information, projects, experience, and details.
